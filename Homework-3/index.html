What is selectors in CSS?

A CSS selectors selects the HTML element(s) you want to style.

  • Selector by name

    Select elements based on name.That's why this text is almost invisible.

  • Selector by ID

    Select elements based on id. That's why this text is white and on the right of the screen.

  • Selector by class

    Select elements based on class. This special class elements should be yellow.

  • Selector by multiple classes

    Select elements based on multiple classes. That's why this text on the center.

  • Selector by nested elements.

    This selector selects all elements inside other elements.That's wht all bold paragraphs will be biege.

  • Attribute selector.

    In this case selector selects all a elements with a href attribute value starting with "http" and make it velvet.

    Ukranian selectors docs

  • Another attribute selector.

    This selector selects all elements with a title attribute that contains a space-separated list of words, one of which is "select" (in our case).

    English selectors docs

  • Adjacent Sibling Selector.

    div > p. Selects p elements where the parent is a div element

  • Another Adjacent Sibling Selector.

    button + p. Selects p elements where the parent is a button element. That's why it's boarder here.

  • Pseudo-class selectors.

    Select elements based on a certain state. In this case - mouse over link will make it bright pink.
